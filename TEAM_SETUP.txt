🚀 QUICK START FOR NEW TEAM MEMBERS

Step 1 - Clone and setup (5 minutes):
  $ git clone <repo-url>
  $ cd big_data_assignment_2
  $ python3.11 -m venv venv
  $ source venv/bin/activate
  $ pip install -r requirements.txt

  Expected output: Test F1 ≈ 0.77-0.80
